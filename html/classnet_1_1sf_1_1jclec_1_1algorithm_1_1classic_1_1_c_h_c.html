Detailed Description

CHC algorithm. This implementation allows non-binary individuals. To do that, user has to define the distance to use in the incest prevention phase, as well as the recombinator to use in the generation phase and the mutator to use in the restarting phase (optional).

Author
Sebastian Ventura

void net.sf.jclec.algorithm.classic.CHC.doUpdate ( )
protectedvirtual

This method includes the following actions:

  • Check whether recombination was successful. If so, return new population
  • If recombination was not successful, update d parameter
  • If d parameter is negative, performs restart
