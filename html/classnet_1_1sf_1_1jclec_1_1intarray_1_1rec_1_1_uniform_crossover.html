void net.sf.jclec.intarray.rec.UniformCrossover.configure ( Configuration  configuration)

Configuration method.

Configuration parameters for UniformCrossover are:

  • []: boolean (default = true)

    If this parameter is set to true</true> individuals will be evaluated after its creation.

  • [-crossover-prob]: double (default = 0.5)

    Locus crossover probability.

  • random-generator: complex

    Random generator used in individuals mutation.

    • random-generator[] String (default 'net.sf.jclec.random.Ranecu')
